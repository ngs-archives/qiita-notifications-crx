Qiita お知らせ
============

Qiita のお知らせをチェックする Chrome 機能拡張

Detailed description
--------------------

Qiita お知らせ は、Qiita の新着情報を Chrome のツールバーアイコンでチェックすることができます。

アイコンをクリックして、通知一覧を表示できます。
